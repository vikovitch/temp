Dit is een simpel html bestandje dat ik heb gemaakt omdat dat kan. Ik zit momenteel op het Firda en ik heb daar onder andere HTML-CSS en JavaScript geleert. Ik heb nog niet heel geavanceerde lessen, maar ik kan iets zoals dit maken uit mijn hoofd.

De Java bestanden die hierbij zitten heb ik gemaakt tijdens mijn zelfstandig leren, ik heb namelijk de W3Schools Java tutorials allemaal doorlopen.

De Python bestandjes zijn twee van een handvol scriptjes die ik met zwaar hulp van ChatGPT heb gemaakt om verschillende redenen. "autokokusen.py" geeft automatich keyboard inputs terwijl ik een toets ingedrukt houd, wat ik had gemaakt om een mechanic in een roblox game maximaal te gebruiken, en "selfprogram.py" speelt een virtuele piano in dezelfde game met vooraf ingeprogrammeerde inputs.

Ik ben best trots op "Button-Clicker", hoewel ChatGPT daar ook de code heeft geschreven. Het gebruikt meerdere modules om met een plaatje van mijn scherm een knop te vinden en dan de muis erop te zetten en te klikken. Ook weer voor een roblox game.

Ik heb nog niet actief geprobeert Python te leren want ik wou eigenlijk wachten tot de school dat doet. Hoewel ik wel actief Java heb zitten leren heb ik het nog niet veel toegepast, dus ik ben nog een beetje een beginner.

Ik ben al zover gekomen ondanks hoe sloom mijn school werkt, omdat ik voor deze opleiding uit passie Luau (Roblox Studio's versie van Lua) heb geleert en veel gebruikt. Je zou kunnen zeggen dat ik "fluent" ben in Lua.
